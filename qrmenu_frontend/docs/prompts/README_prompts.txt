README — Prompts visuels pour MenuHub

But : ce dossier contient les prompts prêts à utiliser dans des outils de génération visuelle (Stitch, Capacity, HeroUI).

Usage :
1. Ouvrir le fichier .txt correspondant à la page désirée.
2. Copier l’intégralité du prompt et le coller dans l’outil visuel.
3. Choisir les viewports recommandés (mobile 390×844, desktop 1440×900).
4. Exporter en PNG/SVG pour la documentation.

Notes :
- Les prompts sont en français et contiennent la palette, le style et la structure compatibles avec l’implémentation actuelle (référez-vous aux fichiers React dans `src/`).
- Après génération, récupérer les valeurs CSS (couleurs, font-sizes, espacements) et injecter dans `premium-theme.css` pour aligner le code.

Fichiers inclus : Home.txt, Menu.txt, MenuItem.txt, ShoppingCart.txt, OrderStatus.txt, Auth.txt, PlacesDashboard.txt, PlaceDetails.txt, QRCodes.txt, Orders.txt, MenuSettings.txt.

Licence/usage : ces prompts sont internes au projet et destinés à la production de mockups pour développement et documentation.
