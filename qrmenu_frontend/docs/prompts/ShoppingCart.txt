Prompt (FR) :
Conçois le panneau panier sous forme de drawer (mobile) et page checkout (desktop) pour MenuHub.

Objectif : résumé clair des articles, totaux, zone notes et validation de commande.

Structure :
- Drawer (mobile) : liste items (miniature + nom + qty control + prix), sous-total, bouton primary sticky "Passer commande" (orange).
- Checkout page : liste détaillée, champ Notes client (textarea), calcul taxes, total final, CTA principal "Valider la commande" (orange) et CTA secondaire "Continuer mes achats" (outline violet).

Interactions/états :
- Désactivation du bouton si panier vide
- Spinner pendant envoi
- Message d'erreur si problème (ex : perte de connexion)

Visuel : cartes sur fond #F5F5F7, totaux en typographie forte, CTA sticky.

Export : PNG mobile + desktop. Fournir valeurs mock pour sous-total, taxes, total.
